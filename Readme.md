Funcionalidades extras añadidas:

Funcionalidad mayor: Se ha agregado un sistema de fidelización de puntos para los clientes, el cual beneficia al cliente con puntos por cada entrada comprada, los cuales puede canjear en futuras compras para obtener un  descuento del precio final de las entradas. Por ejemplo: Comprando una entrada de 10 euros, se obtienen 100 puntos, los cuales puede canjear en futuras compras para obtener un descuento del precio final de las entradas.  Es decir, 100 puntos = 1 euro de descuento.

Otra funcionalidad añadida es que se pueden filtrar por cines segun ciudad y luego por cine dentro de este. Tambien tiene un buscador para buscar por pelicula deseada.
